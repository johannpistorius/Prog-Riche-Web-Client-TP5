Reset Button
	This button lets us reset our canvas.
	However, since we redraw all the lines, when we press the reset button
	and then start drawing again, all the previously erased lines will
	reappear.
Color Picker Button
	This button lets us choose a color.
	We have a similar refresh problem as with the reset button. All the lines will change to the current color.
Size Button
	This button lets us choose a line width.
	We have a similar refresh problem as with the reset button. All the lines will change to the current line size.
Save Button
	This button lets us save our project on our computer.
